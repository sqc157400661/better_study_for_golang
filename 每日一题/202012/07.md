# 子数组的最大累加和问题

## 题目描述

给定一个数组arr，返回子数组的最大累加和

例如，arr = [1, -2, 3, 5, -2, 6, -1]，所有子数组中，[3, 5, -2, 6]可以累加出最大的和12，所以返回12.

[要求]

时间复杂度为O(n)，空间复杂度为O(1)

### 示例1

输入

```
[1, -2, 3, 5, -2, 6, -1]
```

返回值

```
12
```
