# 剑指 Offer 64. 求1+2+…+n

## 题目描述

求 `1+2+...+n` ，要求不能使用乘除法、for、while、if、else、switch、case等关键字及条件判断语句（A?B:C）。

 

### 示例:

```
输入: n = 3
输出: 6

输入: n = 9
输出: 45
```

### 限制：

```
1 <= n <= 10000
```

## 解题思路：

使用递归来实现




## 复杂度分析：

- 时间复杂度 O(n)： 计算 `n+(n−1)+...+2+1` 需要开启n 个递归函数。
- 空间复杂度 O(n)： 递归深度达到 n ，系统使用 O(n) 大小的额外空间。



## 代码：

```
package main

import "fmt"

func sumNums(n int) int {
	ans := 0
	var sum func(int) bool
	sum = func(n int) bool {
		ans += n
		return n > 0 && sum(n-1)
	}
	sum(n)
	return ans
}
func main(){
	fmt.Println(sumNums(3))
}
```
